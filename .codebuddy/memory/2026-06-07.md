- **修复2**: `login.jsp:153` 首页链接 `customerServlet?action=page&pageNo=1` → `customer?pageNo=1`
- **修复3**: `index.jsp` header 补充缺失的"首页"导航链接
- **修复4**: `404.jsp` 三处返回首页链接从 `index.jsp` → `customer?pageNo=1`
